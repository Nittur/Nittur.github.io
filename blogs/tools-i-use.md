# Tools I Use

A running list of things that make my workflow feel less like suffering.

## Editor

**Cursor** — VS Code fork with AI baked in. I was skeptical. Now I can't go back. The tab completion that anticipates multi-line changes is genuinely uncanny.

## Terminal

**Windows Terminal** with PowerShell. I know, I know. But WSL2 is right there when I need Linux tools.

## Notes

Plain `.md` files in a folder. No app, no sync service, no subscription. Just files. This entire website is built on that philosophy.

## Design

- **Figma** for anything visual
- **Excalidraw** for rough system diagrams — the hand-drawn aesthetic removes the pressure of making things perfect

## Music While Working

Lofi hip hop is a cliché for a reason. When I need focus, I put on something without lyrics. Brain treats it like a signal: *we're working now*.

```
Spotify playlist: "lofi beats to debug to"
```

## Browser

Firefox. Chrome is surveillance capitalism in app form.

## This Site

Hand-written HTML, CSS, and vanilla JS. No framework. No bundler. No node_modules folder eating 200MB. The entire site loads in under a second.

---
